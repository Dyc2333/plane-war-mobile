# 飞机大战 - 手机端版本 📱

这是飞机大战游戏的移动端适配版本，专为手机浏览器优化。

## ✨ 新功能

### 虚拟摇杆控制
- **位置**: 屏幕左下区域
- **操作**: 触摸并拖动虚拟摇杆控制飞机移动
- **支持**: 360 度全方向移动

### 虚拟按钮
- **射击按钮**: 屏幕右下角，触摸即可射击
- **炸弹按钮**: 屏幕右上角，使用炸弹道具

### 响应式设计
- 自动适配不同尺寸的手机屏幕
- 横屏/竖屏均支持
- 触摸操作无延迟

## 🎮 操作方法

### 基本操作
1. **移动飞机**: 触摸屏幕左侧区域，拖动虚拟摇杆
2. **射击**: 按住右下角的"射击"按钮
3. **使用炸弹**: 点击右上角的"炸弹"按钮

### 多指操作
- 支持同时使用摇杆和按钮
- 可以同时移动和射击
- 操作流畅无冲突

## 🌐 运行方式

### 方法 1: 直接打开
```bash
# 在本地服务器运行（推荐）
cd 飞机大战手机端
python -m http.server 8000
# 或
npx http-server
```

然后在手机浏览器访问：`http://你的电脑 IP:8000`

### 方法 2: 部署到服务器
将项目上传到任何 Web 服务器即可

### 方法 3: 本地测试
直接用浏览器打开 `index.html`（部分功能可能受限）

## 📱 支持的设备

- ✅ iOS Safari (iPhone/iPad)
- ✅ Android Chrome
- ✅ 其他现代移动浏览器

## 🔧 技术特性

### 触摸优化
- 使用 `touch-action: manipulation` 防止双击缩放
- 触摸事件节流优化
- 多触摸支持（同时控制 + 射击）

### 性能优化
- Canvas 自适应设备像素比
- 根据屏幕尺寸调整特效
- 60 FPS 流畅运行

### 兼容性
- 保留键盘控制（PC 端仍可玩）
- 触摸和键盘可同时使用
- 自动检测设备类型
